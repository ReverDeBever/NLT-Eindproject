# Bio-informatica Website – Eindproduct Module H13

Dit project is gemaakt als eindopdracht voor module Door de ZOETE appel heen bijten.  
Ons gekozen thema is **Bio-informatica**, een vakgebied waarin biologie en informatica worden gecombineerd om DNA-data te analyseren, erfelijkheid te onderzoeken en evolutie te begrijpen.

De website is geschreven in HTML + CSS


---

## Inhoud van de website

De website bevat de volgende onderdelen:

- **Wat is bio-informatica?**  
  Basisuitleg over het vakgebied en waarom het bestaat.

- **Het probleem**  
  Biologen willen erfelijkheid en evolutie onderzoeken, maar de hoeveelheid DNA-data is enorm.

- **De oplossing**  
  Biotechnologie + bio-informatica maken het mogelijk om DNA te lezen, opslaan, vergelijken en interpreteren.

- **DNA, genen en eiwitsynthese**  
  De verplichte biologietheorie uit H13 (DNA → mRNA → eiwit → eigenschap).

- **Hoe werkt bio-informatica?**  
  Van DNA-sequencing tot computeranalyse.

- **Recombinant-DNA-techniek**  
  Wat het is en hoe bio-informatica wordt gebruikt om genen te vinden of te ontwerpen.

- **Toepassingen**  
  Geneeskunde, evolutieonderzoek, landbouw en misdaadoplossing.

- **Opdracht hoofdstuk 12**  
  Wat bio-informatica is en hoe biotechnologie helpt bij misdaden oplossen.

- **Voordelen en nadelen**  
  De impact, risico’s en ethische vragen.
